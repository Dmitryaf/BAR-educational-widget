# BAR Learning Coach

Персональный обучающий LuaUI-виджет для Beyond All Reason.

Проект создаётся не как универсальный помощник и не как средство повышения рейтинга любой ценой. Его задача — помочь одному игроку быстрее освоить базовый цикл BAR:

- поддерживать стабильную энергию;
- расходовать металл;
- соотносить экономику, производство и build power;
- не оставлять заводы и строителей без задачи;
- вовремя расширять добычу;
- постепенно перестать нуждаться в подсказках.

## Структура документации

- `AGENTS.md` — постоянные правила работы для Codex.
- `docs/PRODUCT.md` — назначение продукта и ключевые принципы.
- `docs/LEARNING_MODEL.md` — педагогическая модель подсказок.
- `docs/MVP.md` — точный объём первой версии.
- `docs/ARCHITECTURE.md` — предполагаемая архитектура.
- `docs/NOT_NOW.md` — функции, которые запрещено добавлять в MVP.
- `docs/BAR_API_RESEARCH.md` — шаблон исследования BAR/Recoil API.
- `tasks/PHASE_0_RESEARCH.md` — первая задача для Codex.
- `tasks/PHASE_1_TECHNICAL_SPIKE.md` — следующая задача после исследования.

## Как начать

1. Добавить эти файлы в корень репозитория проекта.
2. Убедиться, что Codex видит `AGENTS.md`.
3. Передать Codex содержимое `tasks/PHASE_0_RESEARCH.md`.
4. Не просить его сразу реализовывать весь MVP.
5. После исследования проверить `docs/BAR_API_RESEARCH.md`.
6. Только затем переходить к `tasks/PHASE_1_TECHNICAL_SPIKE.md`.

## Главная проверяемая гипотеза

Виджет полезен, если он способен:

1. корректно обнаружить устойчивую базовую проблему;
2. показать ненавязчивую подсказку;
3. не повторять её во время исправления;
4. помочь игроку понять причинно-следственную связь;
5. кратко показать результат после матча.

Первая проверяемая проблема — устойчивый дефицит энергии.
