# BAR Learning Coach

Практический LuaUI-тренер первых минут Beyond All Reason.

Вместо набора разрозненных предупреждений виджет ведёт новичка к ближайшему результату одного заранее выбранного opening: запустить доход, закончить первую фабрику, наладить выпуск строителей и боевых юнитов, начать расширение и получить устойчивое T1-состояние.

Подсказка сразу показывает один короткий применимый шаг. Если развитие остановилось из-за подтверждённой проблемы с экономикой, следующий шаг временно заменяется recovery-подсказкой с причиной и вариантами исправления.

Первый учебный сценарий намеренно привязан к конкретным карте, фракции и типу фабрики. Это не универсальный build order и не обещание довести игрока до T2 при любых условиях. Виджет не отдаёт команды, не заполняет очередь и не использует скрытую информацию: все действия остаются за игроком.
