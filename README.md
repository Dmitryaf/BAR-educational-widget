# BAR Learning Coach

Персональный обучающий виджет для Beyond All Reason.

Beyond All Reason сложно осваивать с нуля: игроку нужно одновременно следить за энергией, металлом, производством, строителями и расширением. Из-за этого новичок часто не понимает, почему экономика замедлилась, завод простаивает или ресурсы перестали работать на развитие.

BAR Learning Coach нужен, чтобы во время матча мягко обращать внимание на устойчивые базовые проблемы и помогать понять их причину. Виджет не играет вместо пользователя, не отдаёт приказы юнитам и не предлагает единственно правильный билд. Его задача — поддержать обучение: показать наблюдаемый факт, задать короткий вопрос и помочь игроку самому принять решение.

Цель виджета — постепенно сформировать привычку самостоятельно замечать состояние экономики и производства, чтобы подсказки со временем становились всё менее нужными.
